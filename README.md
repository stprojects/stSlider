# Slider in pure JavaScript (ES6)

Slider call example:

```javascript
let sliderContainer = document.getElementById('your-slider-container');
let opts = {
	slideDuration: 5000,
	items: 1,
	navEnabled: true
}

let sliderInstance = new stSlider(sliderContainer, opts);
sliderInstance.start();
```
